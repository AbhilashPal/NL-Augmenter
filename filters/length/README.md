## length filter

Author: Zhenhao Li

## What type of a filter is this?

This filter filters data with the input text length matching a specified threshold.

Supported conditions:
- greater than: ">"
- less than: "<"
- greater equal to: ">="
- less equal to: "<="
- equal to: "=="
